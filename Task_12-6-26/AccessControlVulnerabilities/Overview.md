## Access Control Vulnerabilities and Privilege Escalation
Access control vulnerabilities occur when an application fails to properly restrict what users are allowed to do. As a result, attackers may gain unauthorized access to data, functions, or resources that should be protected.

## Privilege Escalation
Privilege escalation is a type of access control vulnerability in which a user gains access to privileges or functionality beyond those intended for their account. For example, a normal user may be able to perform actions that are reserved for administrators.
Types of Vulnerabilities That Can Arise with Access Control
Improperly implemented access controls can lead to several vulnerabilities, including:


## Vertical privilege escalation 
  gaining access to higher-level privileges, such as administrator functions.


## Horizontal privilege escalation 
   accessing resources or data belonging to another user with the same privilege level.


## Context-dependent access control vulnerabilities 
   bypassing restrictions based on the user's current state or workflow.
